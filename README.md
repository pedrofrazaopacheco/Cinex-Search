# Cinex Search
A program to find the shortest path between two actors.
It uses the DFS Algorithm

## How to use
- Download the dataset called "basics.tsv.gz" [here](https://datasets.imdbws.com/).
- The data file must be called **data.tsv**
- Run make and ./search
